Vorlesung 4: Chat-Client/-Server und Einführung in PHP (28.04.2025)

Diese Vorlesung drehte sich um die Entwicklung eines Chat-Clients und -Servers. Der Schwerpunkt lag auf der Implementierung eines einfachen Servers und der Gestaltung einer grafischen Oberfläche für den Client.

Server mit Multithreading

Der Server wurde so programmiert, dass er mehrere Clients gleichzeitig bedienen kann. Ein ServerSocket nimmt Verbindungen auf Port 4243 an. Für jeden neuen Client wird ein eigener Thread gestartet.

Die Kommunikation zwischen Server und Clients übernimmt eine innere Klasse namens ClientHandler, die für jede Client-Verbindung zuständig ist. Empfangen Nachrichten werden an alle verbundenen Clients verteilt.

Grafische Oberfläche des Clients

Der Client wurde benutzerfreundlich gestaltet. Eine eigene Klasse verwaltet sowohl die grafische Oberfläche als auch die Netzwerkkommunikation.

  • Die Methode los() erstellt das Fenster mit Chatverlauf, Eingabefeld und Senden-Button.
  • Eine separate Methode stellt die Verbindung zum Server her.
  • Ein Listener reagiert auf den Senden-Button.
  • Ein weiterer Listener sorgt dafür, dass Nachrichten auch per Enter-Taste gesendet werden können.

Übung: Test mit zwei Clients

In einer praktischen Übung wurden zwei Clients lokal gestartet, um die Kommunikation zu testen. Nachrichten wurden erfolgreich über den Server ausgetauscht und korrekt in beiden Fenstern angezeigt.
