Einführung in PHP

Zum Abschluss der Vorlesung erhielten wir eine Einführung in PHP. PHP ist eine serverseitige Skriptsprache, die sich gut für die Entwicklung dynamischer Webseiten eignet.

  • Verarbeiten von Formularen
  • Zugriff auf Datenbanken
  • Erzeugung dynamischer Inhalte auf Webservern

Formularverarbeitung mit $_POST

Es wurde die superglobale Variable $_POST eingeführt, die alle über die POST-Methode gesendeten Formulardaten enthält. Diese Methode eignet sich besonders zur sicheren und nicht sichtbaren Übertragung größerer Datenmengen, etwa in einem Login-Formular.

Ein zentrales Thema war die Datensicherheit bei der Verarbeitung solcher Eingaben. Um Angriffe wie XSS (Cross-Site Scripting) oder SQL-Injection zu vermeiden, ist eine sorgfältige Validierung und Absicherung notwendig.

Einführung in SQL

Es folgte eine Einführung in SQL, die Sprache zur Verwaltung relationaler Datenbanken. Wichtige Befehle sind:

  • SELECT – Daten abrufen
  • INSERT – Daten hinzufügen
  • UPDATE – Daten ändern
  • DELETE – Daten löschen

MySQL und PHP

In der Webentwicklung wird SQL häufig zusammen mit PHP verwendet. Dabei kommt oft MySQL zum Einsatz. Die Verbindung erfolgt über die MySQLi-Erweiterung, die folgende Funktionen bietet:

  • Verbindung zur Datenbank herstellen
  • SQL-Befehle ausführen
  • Ergebnisse verarbeiten

Übung: Formular und Datenbankverbindung

In der Übung wurde ein Formular erstellt, mit dem Name und E-Mail-Adresse eines Benutzers erfasst wurden. Die Daten wurden per POST an ein PHP-Skript übermittelt, dort validiert und in einer MySQL-Datenbank gespeichert.

Diese Aufgabe verdeutlichte die praktische Anwendung der Kombination aus PHP und MySQL in dynamischen Webprojekten.

PHP in der Praxis: Dynamische Inhalte & XAMPP

Mit PHP können Inhalte von Webseiten serverseitig manipuliert werden, bevor die Seite an den Browser des Clients ausgeliefert wird. Dadurch lassen sich dynamische HTML-Seiten generieren.

In der Vorlesung nutzten wir XAMPP 7.4.33-0 zur lokalen Serverausführung.

Übung: Alien-Meldung per Formular

Ziel der Übung war es, ein Formular zu gestalten, über das man einen Bericht über Aliens abgeben konnte. Das Formular wurde über PHP verarbeitet.

Darstellung des Formulars zur Erfassung von Alien-Berichten

Ergebnisanzeige nach dem Absenden

Nach dem Absenden wurde der eingereichte Bericht dynamisch auf einer PHP-Seite angezeigt.

Anzeige des Nutzerberichts nach dem Absenden


    <!DOCTYPE html>
    <html lang="de">
    <head>
      <meta charset="UTF-8">
      <title>Von Aliens entführt – eine Entführung melden</title>
    </head>
    <body>
      <h2>Von Aliens entführt – eine Entführung melden</h2>

      <?php
        $wann_geschah_es = htmlspecialchars($_POST['wanngeschahes']);
        $wie_lange = htmlspecialchars($_POST['wielange']);
        $alien_beschreibung = htmlspecialchars($_POST['beschreibung']);
        $fang_gesehen = htmlspecialchars($_POST['fanggesehen']);
        $email = htmlspecialchars($_POST['email']);

        echo 'Vielen Dank für den Bericht.<br />';
        echo 'Sie wurden ' . $wann_geschah_es . ' entführt und waren ' . $wie_lange . ' weg.<br />';
        echo 'Beschreiben Sie sie: ' . $alien_beschreibung . '<br />';
        echo 'War Fang da? ' . $fang_gesehen . '<br />';
        echo 'Ihre E-Mail-Adresse: ' . $email;
      ?>

    </body>
    </html>
