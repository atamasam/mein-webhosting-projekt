Vorlesung 3: Fortgeschrittene JavaScript-Konzepte (28.10)

Zum Anfang der dritten Vorlesung gab es wieder einen kleinen HTML-Einschub. Hierbei fokussierten wir uns auf das HT (Hypertext) in HTML und haben den <a>-Tag kennengelernt, mit dem auf andere Seiten verlinkt werden kann.

Der <a>-Tag im Detail

Der <a>-Tag, auch als Anker-Tag bekannt, wird verwendet, um Links zu erstellen. Diese Links können auf externe Websites oder interne Seiten innerhalb derselben Website verweisen.

Der grundlegende Aufbau eines Links sieht folgendermaßen aus:

<a href="https://link.de">Linktext</a>

Attribute des <a>-Tags

  • href: Das Attribut href gibt die Ziel-URL des Links an. Ohne dieses Attribut funktioniert der Link nicht.
  • target: Mit diesem Attribut kann festgelegt werden, ob der Link in einem neuen Tab/Fenster geöffnet werden soll (_blank) oder im gleichen Fenster (_self).
  • title: Gibt zusätzliche Informationen über den Link, die beim Hover angezeigt werden.

Links zu internen Seiten

Neben externen Websites können Links auch auf andere HTML-Dateien innerhalb derselben Website verweisen. Dies ist besonders nützlich, wenn man eine mehrseitige Website mit verschiedenen Unterseiten hat. 

<a href="elixiere.html">Elixiere</a>

Empfohlene Ordnerstruktur

Wenn die Website viele Seiten enthält, ist es ratsam, eine Ordnerstruktur anzulegen, um den Überblick zu behalten und die Navigation zu vereinfachen. Dadurch können Links übersichtlicher organisiert werden.

  • Beispiel: Eine Ordnerstruktur für die Website könnte so aussehen:
    • index.html (Startseite)
    • bar/bar.html (Unterseite "Bar")
    • getraenke/getraenke.html (Unterseite "Getränke")
    • images/ (Ordner für Bilder)

Verlinken zwischen verschiedenen Ordnern

Wenn eine Seite aus einem Unterordner auf eine andere Seite in einem anderen Ordner verlinkt, muss der vollständige Pfad angegeben werden. Zum Beispiel könnte ein Link von bar.html auf getraenke.html so aussehen: 

<a href="getraenke/getraenke.html">Getränke</a>

Verlinken auf die übergeordnete Seite

Um von einer Unterseite auf eine Seite im übergeordneten Ordner zu verlinken, kann man das Präfix ../ verwenden. Dieses zeigt an, dass die gewünschte Datei eine Ebene höher im Verzeichnisbaum liegt. 

<a href="../bar.html">Startseite</a>

Wichtige Hinweise für die Verlinkung

  • Achte darauf, dass du bei internen Links immer die Dateiendung angibst, z.B. .html. Andernfalls kann der Browser die Datei nicht finden und lädt sie nicht.
  • Stelle sicher, dass der Pfad korrekt ist. Eine falsche Ordnerstruktur oder ein fehlender Dateiname führt dazu, dass der Link nicht funktioniert.

Zusammenfassung

Der <a>-Tag ist ein grundlegendes und wichtiges Element in HTML, das es ermöglicht, Navigationslinks zwischen Seiten zu erstellen. Durch die richtige Verwendung von Pfaden und Ordnerstrukturen lässt sich die Navigation auf Webseiten optimieren und übersichtlich gestalten.
