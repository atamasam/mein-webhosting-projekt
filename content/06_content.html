Vorlesung 6: DOM-Manipulation, JavaScript-Bibliotheken und Events (25.11)

In dieser Vorlesung haben wir uns mit dem Document Object Model (DOM), JavaScript-Bibliotheken wie jQuery und dem Umgang mit Events in JavaScript beschäftigt. Diese Konzepte sind fundamental, wenn es darum geht, interaktive Webseiten zu erstellen.

Document Object Model (DOM)

Das DOM ist eine strukturierte Darstellung einer Webseite in Form eines Baumes, die es ermöglicht, auf alle Elemente zuzugreifen und sie zu manipulieren. Um Elemente aus dem DOM zu selektieren, nutzen wir Methoden wie:

  • document.getElementById() – Wählt ein Element anhand seiner ID aus.
  • document.getElementsByTagName() – Wählt alle Elemente eines bestimmten Tags aus.
  • document.querySelector() – Wählt das erste Element, das einem CSS-Selektor entspricht.
  • document.querySelectorAll() – Wählt alle Elemente, die einem CSS-Selektor entsprechen.

Das DOM hat jedoch auch seine Herausforderungen: Die Navigation durch den Baum ist nicht immer einfach und zeitaufwendig. Ein weiteres Problem ist, dass es keinen universellen Standard gibt, was dazu führt, dass verschiedene Browser das DOM unterschiedlich interpretieren. Insbesondere der Internet Explorer behandelt Events oft anders und erzeugt teilweise mehr Textknoten als andere Browser.

JavaScript-Bibliotheken

JavaScript-Bibliotheken wie jQuery vereinfachen den Umgang mit häufigen Aufgaben auf Webseiten, wie das Manipulieren von DOM-Elementen, das Hinzufügen von Inhalten oder das Ändern von Attributen. Sie bieten eine Sammlung von Funktionen, die oft viel Code sparen. Diese Bibliotheken werden auf nahezu allen großen Webseiten eingesetzt, einschließlich Microsoft, Twitter und Amazon.

Ein Beispiel für die Einbindung von jQuery:

<script type="text/javascript" src="jQuery.js"></script>

Wichtige Vorteile von jQuery:

  • Geringe Dateigröße (komprimiert nur 30 KByte).
  • Einfacher Umgang für Webdesigner durch CSS-basierte Syntax.
  • Sehr häufig in der Webentwicklung verwendet.
  • Kostenfrei und mit einer großen Entwickler-Community.
  • Enorme Anzahl an verfügbaren Plugins.

Methoden in jQuery

jQuery bietet drei Haupttypen von Methoden:

  • Transparente Methoden: Diese Methoden verändern die Sammlung von Elementen nicht und sind verkettbar.
  • Destruktive Methoden: Diese Methoden verändern die Sammlung und sind ebenfalls verkettbar.
  • Terminierende Methoden: Diese Methoden beenden eine Verkettung und geben kein jQuery-Objekt zurück.

Ein praktisches Beispiel für die Auswahl aller Links auf einer Seite:

var linkList = $('a');

Events in JavaScript

Events wie Mausklicks oder Tastatureingaben sind wichtige Interaktionen auf Webseiten. Wir haben uns mit verschiedenen Maus-Events beschäftigt, die in JavaScript verwendet werden:

Event Beschreibung
click Wird ausgelöst, wenn eine Maustaste gedrückt und wieder losgelassen wird.
dblclick Wird ausgelöst, wenn die Maustaste zweimal schnell hintereinander gedrückt wird.
mouseover Ähnlich wie hover in CSS, wird ausgelöst, wenn die Maus über ein Element bewegt wird.
mouseout Wird ausgelöst, wenn die Maus ein Element verlässt.
mousemove Wird ausgelöst, wenn die Maus bewegt wird.

Beispiel: Event-Listener für einen Button:

<button onclick="function()">Klick mich!</button>

jQuery-Event-Handling

jQuery vereinfacht die Verwaltung von Events. Zum Beispiel, um allen Bildern ein click-Event hinzuzufügen: 

$( 'img' ).click(showFullImg);

Übungen

Die Übungen beinhalteten:

  • Gestaltung kreativer Überschriften: Anpassung von Überschriften mit CSS und jQuery.
  • Automatische Zitate (Pull Quotes): Hervorhebung von Zitaten aus Texten.
  • Akkordeon Register: Erstellen von ein- und ausklappbaren Bereichen für die Strukturierung von Inhalten.

Beispielcode für Akkordeon Register:


$('.accordion').click(function() {
  $(this).next('.content').slideToggle();
});

Zusammenfassung

Zusammenfassend lässt sich sagen, dass das DOM und jQuery sehr hilfreiche Werkzeuge für die Erstellung interaktiver Webseiten sind. Das DOM ermöglicht es, die Struktur einer Seite zu verstehen und zu manipulieren, während jQuery viele der gängigen Aufgaben vereinfacht. Events sind entscheidend für die Interaktivität, und jQuery bietet eine benutzerfreundliche Möglichkeit, mit diesen umzugehen.
