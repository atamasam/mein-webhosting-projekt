Vorlesung 4: Webseitengestaltung mit HTML und CSS (04.11)

Um eine Website zu erstellen, ist es wichtig, die grundlegende Struktur zu planen. Dies kann in Form von Skizzen, sogenannten Wireframes, erfolgen – entweder digital oder auf Papier. Diese Struktur wird dann mit den passenden HTML- und CSS-Elementen umgesetzt.

HTML-Grundlagen

HTML (HyperText Markup Language) ist die Basis jeder Website. Die Sprache besteht aus sogenannten Elementen, die durch Tags definiert werden. Es gibt zwei Haupttypen:

  • Block-Elemente: Diese stehen für sich allein, z. B. <div>, <h1>, <p>.
  • Inline-Elemente: Diese fügen sich in den Textfluss ein, z. B. <q>, <strong>, <img>.

HTML-Tags für Zitate

Es gibt zwei Arten von Zitaten in HTML:

  • <q>: Für kurze Zitate innerhalb eines Absatzes.
  • <blockquote>: Für längere Zitate, die als eigenständiger Block dargestellt werden.

Sonderzeichen in HTML

Um Sonderzeichen wie <, > oder & darzustellen, werden sogenannte Entitäten verwendet:

  • &lt; für < (kleiner als)
  • &gt; für > (größer als)
  • &amp; für & (und-Zeichen)

CSS-Grundlagen

CSS (Cascading Style Sheets) wird verwendet, um das Aussehen einer Website zu gestalten:

  • Selektor: Wählt ein Element aus, z. B. p, h1.
  • Eigenschaften: Definieren das Aussehen, z. B. color oder background-color.

Beispiel:


p {
  background-color: yellow;
  color: blue;
  border: 1px solid black;
}

Externe CSS-Dateien einbinden

Statt CSS direkt in den <style>-Tags zu schreiben, kann eine externe Datei eingebunden werden. Diese wird in der Regel "stylesheet.css" oder "styles.css" genannt. 


    <!-- Im Head-Bereich der HTML-Datei >
    <link type="text/css" rel="stylesheet" href="styles.css">

Beispiel-Inhalt der Datei styles.css:


body {
  font-family: Arial, sans-serif;
  color: #333;
  background-color: #f9f9f9;
}

.highlight {
  color: green;
  font-weight: bold;
}

#unique {
  color: orange;
  text-decoration: underline;
}

CSS-Vererbung

CSS-Eigenschaften werden an untergeordnete Elemente weitergegeben. Dieses Prinzip nennt man Vererbung.

Beispiel:


body {
  color: red;
  font-family: Arial, sans-serif;
}
h1 {
  color: black; /* Überschreibt die Vererbung von body */
}
