Vorlesung 2: Grundlagen der JavaScript-Syntax (21.10)

Willkommen zur zweiten Vorlesung im Modul "Webdevelopment". Diese Einheit behandelt die Grundlagen von JavaScript, einschließlich Entwicklungsumgebungen, Variablen, Datentypen, Kontrollstrukturen, Funktionen und höherer Funktionen.

Was macht der Webbrowser?

  • Der Server „serviert“ die Webseiten und schickt sie an den Browser.
  • Der Browser ruft die Seite ab und zeigt die HTML-Seite an.

HTML

  • HyperText Markup Language: Ermöglicht die Strukturierung von Webseiten.
  • Übersetzt die Tags und den Text in ihnen
  • Sechs Ebenen von Überschriften: <h1> bis <h6>.
  • Aufbau für Elemente = Start-Tag + Inhalt + End_Tag
  • <html> tag gibt an das Dokument in HTML geschrieben ist
  • Kommentare: <!-- Dies ist ein Kommentar -->.
  • Schließende Tags: </tag> helfen dem Browser, den Inhalt klar abzugrenzen.
  • Ausnahme bei Elementen wie <img> (nur ein Tag)

CSS

  • steht für “Cascading Style Sheets”
  • <style> Element um Seiten einen Stil zu verschaffen
  • Style kommt in das <head> Element
  • Benötigt Attribut type (Bsp. type=”text/css”)
  • Syntax: 
    
        p {
        background-color: yellow;
        color: blue;
        border: 1px solid black;
        }

Entwicklungsumgebung

  • Visual Studio Code: Ein Open-Source-Editor, plattformunabhängig und erweiterbar.
  • Alternative Editoren: Atom, Brackets, Notepad++, Bbedit.
  • Node.js: Ermöglicht serverseitiges JavaScript und ist plattformunabhängig.

JavaScript Variablen und Datentypen

  • JavaScript ist dynamisch typisiert: Variablen können Datentypen ändern.
  • Kann zbs. genutzt werden um Textinhalte einer Seite zu verändern oder auch das Layout einer Seite
  • JavaScript-Code in einer eigenen Datei mit <script src=”Verzeichnis/…”> <script>
  • Variablendeklarationen:
    • let: Lokaler Gültigkeitsbereich (empfohlen).
    • var: Globaler oder lokaler Gültigkeitsbereich (veraltet).
    • const: Konstanten, die nur einmal zugewiesen werden können.
  • Datentypen: Number, String, Boolean, Object, null, undefined.

Werte, Typen und Operatoren

  • Arithmetische Operatoren: +, -, *, /.
  • Vergleichsoperatoren: ==, ===, !=, !==.
  • Logische Operatoren: &&, ||, !. 

    let a = 5;
    let b = 10;
    console.log(a + b); // Ausgabe: 15

Kontrollstrukturen

  • if, else, else if: Bedingte Anweisungen.
  • Schleifen: for, while, do-while. 

for (let i = 1; i <=  5; i++) {
    console.log(i); // Ausgabe von 1 bis 5
}

Kontrollausgaben in der Konsole

Verwendung von verschiedenen Konsolenmethoden:

Methode Beschreibung
console.clear() Bereinigt die Konsole.
console.debug() Zeigt Debugging-Informationen an.
console.error() Zeigt eine Fehlermeldung an.
console.info() Zeigt allgemeine Informationen an.
console.log() Zeigt eine Standardnachricht an.
console.trace() Zeigt den Stack-Trace an.
console.warn() Zeigt eine Warnung an.

Arrays

  • Arrays können auf zwei Arten erstellt werden:
    • Mit der Konstrukturfunktion new Array()
    • Mit der Literalsyntax: []
  • Beispiel mit Literalsyntax:
    • 
const fruits = ['Apple', 'Banana', 'Cherry'];
console.log(fruits[0]); // Ausgabe: 'Apple'
  • Methoden für Arrays: push(), pop(), usw.
    • 
fruits.push('Grapes');
console.log(fruits); // Ausgabe: ['Apple', 'Banana', 'Cherry', 'Grapes']

String-Verkettung

  • Strings können mit dem + Operator verkettet werden:
    • 
let greeting = "Good";
let timeOfDay = "Morning";
let result = greeting + " " + timeOfDay; // Ausgabe: "Good Morning"

Objekte

  • Objekte sind Container für Schlüssel-Wert-Paare:
    • 
const book = {
  title: '1984',
  author: 'George Orwell',
  pages: 328,
  read: function() {
    console.log('Reading 1984...');
  }
};
console.log(book.title); // Ausgabe: '1984'
book.title = 'Animal Farm'; // Der Titel des Buches wird geändert

Funktionen

  • Traditionelle Funktionsdefinition:
    • 
function add(a, b) {
    return a + b;
}
  • Pfeilfunktionen (ES6):
    • 
const add = (a, b) => a + b;
  • Pure Functions: Funktionen ohne Seiteneffekte:
    • 
const double = (x) => x * 2;

Höhere Funktionen

Funktionen, die andere Funktionen als Parameter nehmen oder zurückgeben, z. B. map, filter, reduce. 


    const numbers = [1, 2, 3];
    const doubled = numbers.map((n) => n * 2);
    console.log(doubled); // Ausgabe: [2, 4, 6]
